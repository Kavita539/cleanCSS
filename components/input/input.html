Input
Inputs are necessary for user interaction with the website. Inputs are used to take information from the user, and then save the data to the server, accordingly..
Simple Inputs
Simple Input is where user can enter something in the input field and can also havea label present at the top of the input field. They can be customized to show error error-message, and change color accordingly.
Disabled Inputs
Add disabled attribute to your form element to make them disable.
Validator Inputs
These inputs are used for validation or error message.
Strong Password
Minimum 8 characters required!
Radio Inputs
Radio buttons allow the user to select one option from a set.
Checkbox Inputs
Checkboxes allow the user to select one or more items from a set. Checkboxes can be used to turn an option on or off.
