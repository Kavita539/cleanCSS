Badge
Badges are being used to display a notification count or status information. They are small circles, positioned either at top-right or bottom-right of the parent component
Profile Badge
Profile Badge is used to show the current status of a user.
Note : It can be used with 3 variants : online | offline | dnd
Number Badge
Number Badge is used to show the no. of items present in the parent component
