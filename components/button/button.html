Button
Buttons are the soul of an any application. Buttons allow users to take actions, and make choices, with a single tap. You may use a tag or button element, you need to add respective classes, and you are good to go.
Contained Button
Contained buttons have more emphasis, as they use a color fill and shadow. They contain actions that are primary to your app
Outlined Button
Outlined buttons are the ones that do not have hefty background colors. Outlined buttons are used for more emphasis than text buttons due to the stroke.
Icon Button
Icon Buttons can have icon or icon along with text. Use the 'btn' class followed by 'text-icon-btn'.
Link Button
Link Buttons are used to attach a link to the button.
