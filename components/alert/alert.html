Alert
An alert displays a short, important message in a way that attracts the user's attention without interrupting the user's task/flow.
Simple Alert
Simple Alerts can be created using alert class, followed by the variant name. Alerts are available in 4 types- primary alert, success alert, warning alert, error alert.
To use this alert just use class name alert and also add class according to alert type- alert-primary, alert-danger, alert-success, alert-warning. (e.g.class="alert alert-primary").
