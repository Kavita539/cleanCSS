Avatar
Avatar is used to represent a user, and displays the profile picture, initials or fallback icons. They are usually found in circular shapes.
Profile Avatar
Profile Avatars can be created using the class 'avatar', inside the img tag. Available in four different shapes from xs to large.
Text Avatar
You can use initials of user in Avatar as well. You need to include class avatar-txt along with avatar and size class. (e.g. class="avatar avatar-lg avatar-txt")
